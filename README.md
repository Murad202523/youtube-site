# ▶️ YouTube Site

## 📌 Description
A YouTube UI clone built using React and Tailwind CSS.  
The project simulates video platform interface with reusable components.

## 🚀 Features
- Video grid layout  
- Sidebar navigation  
- Search functionality UI  
- Responsive design  

## 🛠 Tech Stack
- React  
- JavaScript (ES6+)  
- Tailwind CSS  

## ▶️ How to Run
1. Clone the repository  
2. Run:
```bash
npm install  
npm run dev
